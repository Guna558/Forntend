To establish a Network we require  Hardware => physical components  Software => is a group of computer programs  Protocol => A protocol represents specification of rules to be followed by ever computer in the network. Protocol consists of  When to send /receivone data  How to send / receive data Client: A client is a computer that sends a request. Server: A computer which responds to the client requests and gives response. Software: Is a group of computer programs that drives the hardware to perform the task. Computer program: Is sequence of instructions that a computer can interpret and execute, When we execute any computer program, it reads the source code, converts the source code to a machine level language and the machine level code is read by the processor and runs the operation as instruction given in the program.
